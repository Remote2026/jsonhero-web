交互流程

进入首页直接看到三栏布局，中间是输入区，左右栏显示空状态提示
粘贴或拖入 JSON无需跳转页面，中间输入区直接变成 JSON 列视图
点击示例加载示例 JSON，同样在当前页面渲染查看器
分享链接提交后生成 `/j/$id` 链接，用户可分享

修改思路

  • index.tsx 不再走营销首页 — 而是复用 /j/$id 的三栏布局，但初始状态不加载 JSON
  • 输入区作为 JsonView 的"空状态" — 类似 :empty 或 loading state
  • 不跳转页面 — 粘贴/拖入后同一页面内从输入态切换到查看态，用 replaceState 更新 URL
  • header 简化 — 不需要 "Try now" 按钮了，因为已经在用了
